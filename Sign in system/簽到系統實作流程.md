# 線上報到手寫簽名檔系統 — 完整實作流程

## 📋 系統架構

```
學員掃 QR Code (GitHub Pages)
         ↓
簽到頁面：填表 + 手寫簽名
         ↓
簽名圖片 → Firestore (Firebase)
文字資料 → Google Sheets (Apps Script)
         ↓
管理後台：查看所有簽到 + 簽名縮圖
```

---

## 🔧 第一步：準備工作

### 1.1 GitHub 設定
- 建立一個新 repo，例如：`sign-in-system`
- 啟用 GitHub Pages：Settings → Pages → Source: main branch
- 記下 GitHub Pages URL：`https://[username].github.io/sign-in-system/`

### 1.2 Firebase 專案
- 到 https://console.firebase.google.com
- 建立新專案，例如：`signinsystem2026`
- 啟用 Firestore Database（選擇測試模式或自訂規則）
- 啟用 Authentication（使用匿名登入或不登入）
- 建立兩個 Web 應用：一個用於**簽到頁**，一個用於**管理後台**
- 取得 Firebase Config（apiKey, authDomain, projectId 等）

### 1.3 Google 帳戶
- 建立 Google Apps Script 專案
- 建立 Google Sheets 試算表（將被 Apps Script 寫入）
- 記下 Apps Script 部署後的 URL

## 📄 第二步：建立簽到頁面（index.html）

**檔案位置：** 根目錄 `index.html`

**核心功能：**
- 表單：姓名、單位、職稱、電話、身分證末 4 碼
- Canvas：手寫簽名（支援觸控 / 滑鼠）
- Firebase：簽名圖片上傳到 Firestore（base64）
- Google Apps Script：文字資料送到 Sheets
- 成功頁：顯示 3 秒後自動重置

**關鍵步驟：**

1. **嵌入 Firebase SDK：**
   ```html
   <script type="module">
   import { initializeApp } from 'https://www.gstatic.com/firebasejs/11.8.1/firebase-app.js';
   import { getFirestore, collection, addDoc } from 'https://www.gstatic.com/firebasejs/11.8.1/firebase-firestore.js';
   
   const firebaseConfig = {
     apiKey: "YOUR_API_KEY",
     authDomain: "YOUR_AUTH_DOMAIN",
     projectId: "YOUR_PROJECT_ID",
     storageBucket: "YOUR_STORAGE_BUCKET",
     messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
     appId: "YOUR_APP_ID"
   };
   
   const app = initializeApp(firebaseConfig);
   const db = getFirestore(app);
   </script>
   ```

2. **Canvas 簽名捕捉：**
   - 在 HTML 中加 `<canvas id="signatureCanvas"></canvas>`
   - JS 監聽 `mousedown`, `mousemove`, `mouseup`, `touchstart`, `touchmove`, `touchend`
   - 用 `canvas.toDataURL('image/png')` 轉成 Base64

3. **表單驗證：**
   - 檢查必填欄位（姓名、單位、職稱）
   - 檢查簽名不為空（`canvas.toBlob()` 檢查像素）

4. **雙段上傳：**
   
   **第一步：上傳簽名到 Firestore**
   ```js
   const sigData = canvas.toDataURL('image/png');
   const docRef = await addDoc(collection(db, 'signatures'), {
     name: formData.name,
     signature: sigData,
     time: new Date().toISOString()
   });
   const sigDocId = docRef.id;
   ```
   
   **第二步：送表單 + docId 到 Apps Script（GET）**
   ```js
   const params = new URLSearchParams({
     name: formData.name,
     org: formData.org,
     jobTitle: formData.jobTitle,
     phone: formData.phone,
     idLast4: formData.idLast4,
     sigDocId: sigDocId,
     time: new Date().toLocaleString('zh-TW', {timeZone: 'Asia/Taipei'})
   });
   new Image().src = SCRIPT_URL + '?' + params.toString();
   ```

5. **成功頁（3 秒倒計時）：**
   ```js
   function showSuccess(name) {
     let sec = 3;
     const interval = setInterval(() => {
       sec--;
       if (sec <= 0) {
         clearInterval(interval);
         resetForm();
       }
     }, 1000);
   }
   ```

## 📊 第三步：Google Apps Script 後端

**部署方式：**
1. 開啟 Google Sheets
2. 工具 → 指令碼編輯器
3. 貼入以下代碼

```js
const SHEET_NAME = '簽到記錄';

function doGet(e) {
  if (e.parameter.action === 'getData') return getSheetData();
  if (e.parameter && e.parameter.name) {
    try { saveRecord(e.parameter); } catch(err) { console.error(err); }
  }
  return ContentService.createTextOutput('ok').setMimeType(ContentService.MimeType.TEXT);
}

function saveRecord(p) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sheet = ss.getSheetByName(SHEET_NAME);
  if (!sheet) {
    sheet = ss.insertSheet(SHEET_NAME);
    sheet.appendRow(['#','簽到時間','姓名','服務單位','職稱','聯絡電話','身分證後4碼','活動名稱','簽名DocId']);
    sheet.setFrozenRows(1);
    const hdr = sheet.getRange(1, 1, 1, 9);
    hdr.setBackground('#2d6a50');
    hdr.setFontColor('#ffffff');
    hdr.setFontWeight('bold');
  }
  sheet.appendRow([
    sheet.getLastRow(),
    p.time || new Date().toLocaleString('zh-TW', {timeZone: 'Asia/Taipei'}),
    p.name || '', p.org || '', p.jobTitle || '',
    p.phone || '', p.idLast4 || '', p.event || '',
    p.sigDocId || ''
  ]);
}

function getSheetData() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName(SHEET_NAME);
  if (!sheet || sheet.getLastRow() <= 1) {
    return ContentService.createTextOutput('[]').setMimeType(ContentService.MimeType.JSON);
  }
  const data = sheet.getDataRange().getValues();
  const headers = data[0];
  const rows = data.slice(1).map(row => {
    const obj = {};
    headers.forEach((h, i) => { obj[h] = row[i]; });
    return obj;
  });
  return ContentService.createTextOutput(JSON.stringify(rows)).setMimeType(ContentService.MimeType.JSON);
}
```

**部署步驟：**
1. 按「部署」→ 「新部署」
2. 選擇類型：「網路應用程式」
3. 執行身分：選擇你的帳號
4. 存取：「任何人」
5. 複製部署 URL，貼到 `index.html` 的 `CONFIG.scriptUrl`

## 🔐 第四步：Firestore 安全規則

**路徑：** Firebase Console → Firestore → Rules

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // 允許所有人讀寫 signatures（簽到時使用）
    match /signatures/{document=**} {
      allow read, write: if true;
    }
  }
}
```

發佈規則。

## 🖥️ 第五步：管理後台（admin.html）

**檔案位置：** 根目錄 `admin.html`

**核心功能：**
- 從 Google Sheets 讀取文字資料
- 從 Firestore 讀取簽名圖片（用 docId 查詢）
- 顯示簽名縮圖，點擊放大
- 搜尋、篩選、匯出 CSV

**關鍵步驟：**

1. **讀取 Sheets 資料：**
   ```js
   async function loadFromSheets() {
     const res = await fetch(SCRIPT_URL + '?action=getData');
     const json = await res.json();
     return json.map(r => ({
       time: r['簽到時間'],
       name: r['姓名'],
       sigDocId: r['簽名DocId']
     }));
   }
   ```

2. **從 Firestore 讀簽名：**
   ```js
   async function loadSig(docId) {
     if (!docId) return null;
     try {
       const snap = await getDoc(doc(db, 'signatures', docId));
       return snap.exists() ? snap.data().signature : null;
     } catch(e) { return null; }
   }
   ```

3. **非同步載入簽名（不阻擋表格）：**
   ```js
   records.forEach(async (r, i) => {
     if (!r.sigDocId) return;
     const sig = await loadSig(r.sigDocId);
     if (sig) updateSigCell(i, sig);
   });
   ```

## 🔍 常見問題排查

| 問題 | 原因 | 解決 |
|------|------|------|
| 簽名上傳失敗 | Firebase SDK 未載入 | 檢查 `firebaseConfig` 是否正確 |
| 簽名不顯示在後台 | sigDocId 未存到 Sheets | 檢查 Apps Script GET 參數是否完整 |
| 管理後台 CORS 錯誤 | 未用 localhost 打開 | 用 `python -m http.server` 啟動伺服器 |
| Firestore 讀不到簽名 | 安全規則限制 | 檢查規則是否允許 `allow read, write: if true;` |
